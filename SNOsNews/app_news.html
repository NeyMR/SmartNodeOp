SmartNodeOps - Release Notes

Version 1.0.4.6 Release Highlights:

⚠️Remember Downloading this tool From Unauthorized Sources May Compromise the Security of your Node!⚠️

Pi Node Installer:

  • One-click installation of Virtual Machine Platform and Windows Subsystem 2 (WSL2).
  • Automatic installation of required software, including Docker 4.12 and the latest Pi Node App (v0.4.5).

Json File Fix:

  • Intelligent repair of Json file issues, automatically resolving Node Not Loading problems commonly encountered in new Pi Node installations.

Port Check and Windows Firewall Opener:

  • Ensures your Pi Node host machine has the required open ports for optimal Pi Node performance.
  • Provides step-by-step instructions to open necessary ports if found closed.

Pi Node Metrics:

  • Comprehensive display of valuable node information for efficient diagnosis and monitoring.
  • Access to error logs and other vital metrics.
